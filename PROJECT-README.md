\## Development rules



1\. La web debe ser multi idioma, ingles español donde Ingles es por defecto(Los botones de idioma deben de estar en la barra de navegación y deben ser simples y minimalistas)



2\. incluye un slidershow o carousel de imágenes con efectos css modernos.



3\. renombra las imágenes de la carpeta images para un eficiente metadatos o posicionamiento web.



4\. Incluye los metadatos multi idioma para este tipo de negocio.



5\. No incluyas formulario de contacto.



6\. Incluye un botón de subir arriba y un botón de whatsapp para el contacto.



7\. Incluye botones de redes sociales.



8\. Barra de navegación: inicio, quienes somos, servicios, portafolio, contacto.



9\. Incluye mapa Google maps.



10\. En Portafolio has que cada imagen se amplie en un modal para que se vea en grande.



11\. Incluye el logotipo de la carpeta images.
